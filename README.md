# Tareasia - Descubre Todo Sobre Tu Schnauzer

## Descripción

Tareasia es una aplicación web interactiva diseñada para amantes de los perros Schnauzer. Permite a los usuarios subir fotos de sus mascotas y obtener información detallada sobre la raza Schnauzer Miniatura, incluyendo características, cuidados específicos y su historia.

La página está construida con tecnologías web básicas (HTML, CSS y JavaScript) y ofrece una experiencia intuitiva para explorar y aprender sobre esta fascinante raza de perro.

## Características

- **Subida de Fotos**: Interfaz drag-and-drop para subir imágenes de tu Schnauzer.
- **Información de la Raza**: Detalles sobre el Schnauzer Miniatura, incluyendo su temperamento y características físicas.
- **Cuidados Específicos**: Consejos sobre cepillado, ejercicio y dieta.
- **Historia de la Raza**: Origen y evolución del Schnauzer desde Alemania.
- **Interfaz Responsiva**: Diseñada para funcionar en dispositivos móviles y de escritorio.

## Tecnologías Utilizadas

- **HTML5**: Estructura de la página.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Funcionalidad interactiva para la subida de archivos y manipulación del DOM.

## Cómo Usar

1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` en tu navegador web preferido.
3. Haz clic en "Subir Foto" o arrastra una imagen de tu Schnauzer al área designada.
4. Explora la información proporcionada sobre la raza.
